# 📱 WhatsApp Templates — DOAGuru InfoSystems

This file contains the complete documentation for the three WhatsApp templates.
These must be registered in the Meta Business Manager.

> **IMPORTANT:** WhatsApp Cloud API only allows sending pre-approved templates.
> You must create and approve each template below in your Meta Business Manager > WhatsApp Manager > Message Templates.

## Template 1: Proposal Sent Notification

### When is it sent?

Whenever a Proposal is shared with a Client —

- **To Client:** With the link to review and approve the proposal.
- **To Admin/Owner:** A notification that the proposal has been sent to the client.

## Template 2: Periodic Pending Payment Summary

### When is it sent?

**Every 5 days at 10:30 AM**, exclusively to the **Admin/Owner**.
This message calculates the overall "Total Pending" and displays the list of ONLY those clients whose pending amount is greater than zero. Clients with zero dues are ignored.

**Template Name (Register in Meta):** `periodic_payment_summary`
**Language:** `en`
**Category:** `UTILITY`

```
📊 *[REPORT] Pending Payments*
━━━━━━━━━━━━━━━━━━━━━━━━
🗓️ Date: {{1}}

Summary of clients with pending payments:

💰 *Total Outstanding: ₹{{2}}*

📋 *Client-wise Breakdown:*
{{3}}

━━━━━━━━━━━━━━━━━━━━━━━━
Please follow up with the clients listed above for payments.

*— DOAGuru Automated Report*
```

**Variables Mapping:**
| Variable | Value |
|----------|----------------------------------------------|
| {{1}} | Today's Date (DD/MM/YYYY) |
| {{2}} | Total Pending Amount (e.g., 1,25,000) |
| {{3}} | Client-wise list (formatted multi-line text) |

**Format example for `{{3}}` (generated by code):**

```
• Abhinav Panday (ID: 42)
  Pending: ₹22,288 | Invoice: INV-154

• Rahul Sharma (ID: 67)
  Pending: ₹15,000 | Invoice: INV-161
```

> **NOTE:** If there are many clients, the list will be split into chunks of 10 per message.

## ⚙️ Steps to Register Templates in Meta Business Manager

1. **Login:** https://business.facebook.com
2. **Navigate:** `WhatsApp Manager` > `Message Templates` > `Create Template`
3. **Choose Category:** UTILITY or MARKETING (as specified above)
4. **Template Name:** Enter exactly as given above (lowercase, underscores)
5. **Language:** English
6. **Message Body:** Copy and paste the text provided above (keep `{{1}}`, `{{2}}`, etc. exactly as they are)
7. **Submit for Review** — Meta usually takes 24 to 72 hours for review.
